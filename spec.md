# Specification

## Summary
**Goal:** Add a static 100-star rating display in the header's upper-right corner, near the existing wishlist button.

**Planned changes:**
- Add a static star rating widget to the Header component showing "100/100" (e.g., "★ 100/100") with a "Rating" label
- Position the widget in the upper-right area alongside the wishlist button without overlapping it
- Style the widget using the existing blue-crystal-moon brand palette and font styles
- Ensure the widget is visible on both mobile and desktop layouts

**User-visible outcome:** Visitors see a "★ 100/100" rating badge in the top-right corner of the header on all pages, styled consistently with the rest of the site.
